- 自动播放/暂停（3秒间隔）
- 缩略图快速跳转

## 🚀 快速开始

### 1. 准备内容

#### 方法一：使用图片（推荐）
1. 将您的画册页面保存为 JPG/PNG 图片
2. 按顺序命名为：`cover.jpg`, `page-1.jpg`, `page-2.jpg`, ..., `back.jpg`
3. 放入 `pages/` 目录

#### 方法二：从PDF转换
使用在线工具将PDF转为图片：
- https://smallpdf.com/cn/pdf-to-jpg
- https://www.ilovepdf.com/zh-cn/pdf_to_jpg

### 2. 本地预览

双击打开 `index.html` 即可在浏览器中预览，或使用本地服务器：

```bash
cd yunzhan-flipbook

# Python
python -m http.server 8080

# Node.js
npx serve

# PHP
php -S localhost:8080
```

访问 `http://localhost:8080`

### 3. 部署到免费平台

#### GitHub Pages（推荐）
1. 创建 GitHub 仓库
2. 上传所有文件
3. Settings → Pages → Branch: main, Folder: /
4. 获得链接：`https://用户名.github.io/仓库名/`

#### Vercel
1. 访问 vercel.com
2. 导入仓库
3. 一键部署
4. 获得链接：`https://xxx.vercel.app`

## 🔧 自定义配置

### 修改翻页速度
编辑 `js/main.js`，找到：
```javascript
flippingTime: 600, // 翻页动画时间（毫秒）
```
- 更快：改为 `400`
- 更慢：改为 `800`

### 修改自动播放间隔
编辑 `js/main.js`，找到：
```javascript
}, 3000); // 3秒间隔
```
改为其他毫秒数

### 修改主题颜色
编辑 `css/style.css`：
```css
/* 背景色 */
body {
    background: #2a2a2a; /* 深灰色 */
}

/* 高亮色 */
.tool-btn:hover {
    color: #4CAF50; /* 绿色 */
}
```

## 🐛 常见问题

### Q: 图片不显示怎么办？
A: 检查 `pages/` 目录中是否有对应文件名的图片；确保文件名正确（区分大小写）

### Q: 翻页不流畅？
A: 压缩图片大小（建议单张<500KB）；降低 `flippingTime` 参数

### Q: 移动端无法滑动？
A: 确保使用了 viewport meta 标签；检查浏览器是否支持触摸事件

### Q: 如何添加更多页面？
A: 
1. 在 `index.html` 的 `#flipbook` 内添加新的 `.page` div
2. 将对应图片放入 `pages/` 目录
3. 刷新页面

## 📊 性能优化建议

1. **图片压缩**：使用 TinyPNG 压缩到 500KB 以内
2. **懒加载**：对于大量页面，实现按需加载
3. **CDN加速**：使用 CDN 托管静态资源
4. **缓存策略**：配置浏览器缓存
